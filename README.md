# 混沌创新技能

混沌创新技能是为你的 AI 助理置入混沌创新商业智库，帮助你更好地完成“混沌课程学习”和“商业问题拆解”。

它适合两类典型场景：

- 你已经明确要学混沌课程，想搜课、筛课、搭学习路径、读课程文稿、提炼方法论
- 你在面对一个真实商业问题，想先判断方向、拆解问题、重构卖点、定价、找新机会

---

## 技能介绍

混沌创新技能当前主要提供两类能力：

1. 课程推荐与学习
2. 实用创新工具

### 1. 课程推荐与学习

当你明确提到混沌课程、学习路径、搜课、读课、课程体系、课程文稿时，技能会优先进入课程学习模式，帮你做这些事：

- 搜索相关课程
- 从候选课程里筛出最适合的 1 到 3 门
- 按你的问题搭一个学习顺序
- 读取单门课程文稿并提炼重点
- 把课程里的方法论和步骤整理成你能直接拿去用的结论

你可以这样问：

- `我要做出海，应该学习哪些混沌课程？`
- `推荐 3 门和 AI 转型最相关的混沌课程`
- `帮我系统学混沌里跟品牌有关的课程`
- `这门课具体讲了什么？`
- `把这门课里的实操步骤整理成我可以直接执行的清单`

### 2. 实用创新工具

当你没有明确说要课程，而是希望先把一个商业问题拆开、重构、澄清或找到方向时，技能会优先用混沌创新工具来处理。

当前已完成的 5 个实用创新工具是：

- 一句话价值提亮器
- 创意点子爆破器
- 产品卖点创新器
- 精准定价器
- 以小博大探测器

你可以这样问：

- `我们是做 AI 知识库的，帮我把一句话价值主张说清楚`
- `我的产品太同质化了，帮我找新的卖点方向`
- `我不知道怎么定价，帮我重构一下价格逻辑`
- `我想切入一个热门市场，但不想跟大公司正面竞争，帮我找切口`
- `我想找新的业务方向和组合机会`

## 如何配置、使用

### 1. 获取密钥

访问下面的页面登录并新建 API Key：

[https://tools.hundun.cn/h5Bin/aia/#/keys](https://tools.hundun.cn/h5Bin/aia/#/keys)

你需要的是 `hd_sk_` 开头的密钥。

### 2. 配置方式

首次运行需要鉴权的脚本时，如果本地还没有配置文件，系统会自动创建：

- `~/.hdxy_config`

默认会写入线上接口地址：

- `https://hddrapi.hundun.cn`

如果你是在和支持该技能的 AI 助手对话，通常不需要手动改文件，直接把 `hd_sk_` 开头的密钥发给 AI 即可。

### 3. 基本使用流程

1. 获取密钥
2. 把密钥提供给 AI 助手或写入配置
3. 直接描述你的问题
4. 技能会自动判断应该走课程学习，还是创新工具
5. 如果需要课程文稿或真实搜课，技能会调用对应接口继续完成

### 4. 常见使用方式

#### 搜课

```bash
./scripts/search_courses.sh "关键词"
```

#### 查课程体系

```bash
./scripts/get_trees.sh
./scripts/get_courses_by_tree.sh "体系ID"
```

#### 看文稿版本

```bash
./scripts/get_script_version.sh "课程ID"
```

#### 读取课程文稿

```bash
./scripts/get_script.sh "课程ID"
```
