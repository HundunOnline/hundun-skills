# 创意点子爆破器

当用户想找新方向、新产品、新服务、新组合，但思路被行业惯性卡住时，使用这个流程。

## 适用问题

- “我想做个新东西，但不知道从哪里开始想”
- “这个行业还能有什么新机会”
- “我们现在只会模仿，怎么找到新点子”

## 不适用问题

- 用户已经锁定单一方案，只想做执行排期
- 用户需要严肃的财务测算、法务判断或投融资建议

## 最少输入

- 当前想探索的领域或问题
- 现有资源、能力、渠道中最能用的 1 到 3 项
- 最想避免的约束或代价

如果缺信息，最多补 3 个问题：

1. 你现在最想解决的具体问题是什么
2. 你手上已经有的资源或能力是什么
3. 你不想碰的边界是什么

## 执行流程

1. 先界定“用户真正想解决的问题”，不要被表面的行业名词带跑
2. 把问题拆成最小要素：目标人群、关键场景、当前替代方案、关键限制
3. 用跨界迁移、重组、删减、反转这几种方式打散既有解法
4. 结合第二曲线和第一性原理，优先寻找：
   - 被忽视的非主流人群
   - 被默认绑定但其实可以拆开的价值链环节
   - 能用现有能力快速试的小切口
5. 产出 3 到 5 个不同方向的点子，不要只给一个答案
6. 对每个点子补一句“为什么它不是旧思路换皮”
7. 最后选出最值得先试的 1 个方向，并说明先验证什么

## 输出结构

1. 用户当前问题的真正切口
2. 3 到 5 个新点子方向
3. 每个方向的核心逻辑
4. 最推荐先试的方向
5. 下一步最小验证动作

## 当前版本的方法要求

- 优先追求“方向增量”，不是一次性给到完整商业计划
- 第二曲线里优先使用这些具体模型：
  - `分形创新`：从现有能力里长出多个小方向
  - `组合创新`：拆开产品、市场、资源、组织后重新组合
  - `边缘分化`：优先找边缘人群、边缘场景和侧枝机会
  - `破界创新`：当行业默认假设本身有问题时，直接换题
- 使用前直接到 [第二曲线创新.md](../foundations/第二曲线创新.md) 里找对应章节并补读
- 如果需要明确按第一性原理展开，先追加阅读 [第一性原理.md](../foundations/第一性原理.md)
- 每个点子都要能说清楚新在哪里、为什么值得试
- 不要把 brainstorming 变成空泛发散，必须落回用户现有资源和现实约束
